# Expence-Tracker
Project Title
Smart Expense Tracker

DESCRIPTION
A simple web app that helps users track daily expenses by category. Data is saved in the browser using Local Storage.

FEATURES
Add expenses
Delete expenses
Filter by category
Persistent storage (Local Storage)
Responsive design

TECH USED
HTML
CSS
JavaScript (Vanilla JS)
Local Storage API

HOW TO USE
-Enter expense name and amount
-Choose category
-Click add
-Expenses appear in list and are saved automatically

FUTURE IMPROVEMENTS
-Charts
-Monthly summary
-Currency selector
